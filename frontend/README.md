# (UN)official Catasto Grotte - Frontend

This is the React-based frontend for the Catasto Grotte project.

## 🛠 Tech Stack

- **Framework:** React 18+
- **Build Tool:** Vite
- **Language:** TypeScript
- **Styling:** Tailwind CSS v3
- **Maps:** Leaflet

## 🚀 Key Pages

- `Landing.tsx`: Project overview.
- `CaveList.tsx`: Searchable table of caves.
- `CaveMap.tsx`: Interactive map with clustering.
- `CaveDetail.tsx`: Individual cave data and media.
- `Dashboard.tsx`: Administrative overview.
- `CaveForm.tsx`: Create/Edit interface for cave data.

## 🏁 Running Locally

The frontend is intended to be run via Docker Compose from the root directory.

```bash
# From the project root
docker compose up frontend
```

For more detailed instructions, please refer to the [Root README](../README.md).
